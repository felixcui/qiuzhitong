## 🛠️ 优化工具与资源

### 使用的工具
- **ui-ux-pro-max** - UI/UX 设计智能工具
  - 50+ 设计风格
  - 21 种色彩方案
  - 50 种字体组合
  - 99 条 UX 指南
  - 25 种图表类型
  - 支持 13 种技术栈

### 相关命令
```bash
# 设计系统分析
python3 skills/ui-ux-pro-max/scripts/search.py "学术 理论 导航" --design-system -p "求知汇"

# UX 指南搜索
python3 skills/ui-ux-pro-max/scripts/search.py "academic navigation" --domain ux

# Tailwind 最佳实践
python3 skills/ui-ux-pro-max/scripts/search.py "card hover animation" --stack html-tailwind
```
